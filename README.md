# 📚 AI PDF Research Assistant

An AI-powered PDF Research Assistant built with Streamlit, LangChain, ChromaDB, Hugging Face Embeddings, and Google Gemini.

## Features

- 📄 Upload PDF
- 🧠 AI-powered Question Answering
- 🔍 Semantic Search
- 💾 Chroma Vector Database
- 🤖 Google Gemini Integration
- 💬 Chat Interface

## Tech Stack

- Python
- Streamlit
- LangChain
- ChromaDB
- Hugging Face
- Google Gemini

## Installation

```bash
git clone <repository-url>
cd AI-PDF-Research-Assistant

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

streamlit run app.py
